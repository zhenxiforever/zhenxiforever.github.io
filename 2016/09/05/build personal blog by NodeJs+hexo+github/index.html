使用NodeJs+hexo+github搭建个人博客

本文详细描述如何使用hexo在github上搭建属于自己的博客[Windows和linux(Centos)]上；

名词

hexo

hexo

Nodejs

Nodejs

github

Windows 搭建步骤

环境安装

1. 安装配置git
由于这里只是搭建个环境，不需要其他的一些工具，简单的安装 GitForWindows 就行了，需要注意的时，在安装时选择 Run Git from the windows command Prompt，这样在Windows的命令行可以直接使用git命令;

  • 配置用户名和邮箱 
deploy:
  type: git
      repository: ssh://git@github.com/zhenxiforever/zhenxiforever.github.io.git
 branch: master

然后修改_config.yml的theme项为yilia就可以了。

  • 添加新文章。运行命令hexo n "filename"会在source/_posts/filename.md，编辑该md文件后，运行hexo g和hexo d命令就可以在github上看到你的新文章;
